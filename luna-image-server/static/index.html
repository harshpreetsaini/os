🖼️ Luna Image Server

MPC (Model Context Protocol) Server for Image Operations

Status: Running

📡 Available Endpoints

GET
/api/image/info - Get image metadata
Retrieve detailed information about the main image (filename, format, dimensions, size)
GET
/api/image/view - View full-size image
Display the full-size version of the main image
GET
/api/image - Get thumbnail
Retrieve a smaller thumbnail version (300x300 max) of the main image
GET
/api/image/info/<filename> - Get metadata for uploaded image
Get metadata for images uploaded to the static directory
GET
/api/images - List all uploaded images
Get a list of all images in the static directory with their metadata
POST
/api/image/upload - Upload an image
Upload an image file (PNG, JPG, JPEG, GIF, BMP, SVG) to the static directory
POST
/api/process/image - Process an image
Process images with operations like blur, resize, crop, filters, AI effects
GET
/api/search/images - Search images
Search images by properties like format, size, filename pattern
POST
/api/compare/images - Compare two images
Compare two images and return similarity metrics and comparison data
